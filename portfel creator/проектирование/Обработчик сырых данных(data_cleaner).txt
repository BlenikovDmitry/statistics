Обработчик сырых данных(cleaner)

Принимает на вход данные из файла с сырыми данными из файла raw.csv
Записывает очищенные данные в файл clear.csv

Сырые данные: таблица облигаций с сайта ммвб

Функционал:


1) Считать все данные из файла(read_data) файл raw.csv
2) Вычленить данные(make_clear_data):

- наименование
- код
- цена
- доходность последней сделки
- дата погашения
- непогашенный долг(номинал)


3) записать в файл (write_clear_data) файл clear.csv
