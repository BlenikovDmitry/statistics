Сервис получения данных с сайта ЦБ (cbr.ru)

Есть следующие команды (вводятся в адресной строке):
{y1} — год начала наблюдений и {y2} — год конца наблюдений

Список команд:

  • Процентные ставки по депозитам физлиц: /deposit/{y1}/{y2}
  • Процентные ставки по ипотеке: /mort/{y1}/{y2}
  • Процентные ставки по кредитам физлиц: /credit/{y1}/{y2}
  • Задолженность по кредитам физлиц: /credit_debt/{y1}/{y2}
  • Просроченная задолженность по кредитам физлиц: /credit_overdue_debt/{y1}/{y2}
